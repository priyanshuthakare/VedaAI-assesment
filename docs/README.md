# VedaAI Assessment Creator

Welcome to the VedaAI Full Stack Engineering Assignment documentation.

This repository implements a complete, production-ready AI Assessment Creator built to match the provided Figma specifications. It allows educators to generate highly structured question papers dynamically using AI, complete with background job processing, real-time WebSocket updates, and server-side PDF generation.
